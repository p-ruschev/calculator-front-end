Hello, this is simple calculator application.

Using this app you can calculate mathematical expressions. Logged users (for creating an account you need to provide an email which does not need to be real email, only following the same syntax as real one) could save their expressions for later use. While saving an estimation you could add brief description, which you can use to search your savings by. This calculator has the ability to create and save formulas with variables which you can substitute with different numbers, so that the formula in use could be applied in different cases.
You could create create an account or log into log into one if you have such or go straight to the calculator in order to try it.
